Zoned is a mobile app designed to enhance the classic game of hide and seek by providing real-time location tracking and customizable game zones, making it easy and fun to play with friends in various environments!

## Features

- Real-time location tracking
- Customizable game zones

## Getting Started

1. Clone the repository
2. Ensure [Flutter](https://flutter.dev/docs/get-started/install) is installed and added to your system PATH.
3. Install dependencies:
    ```sh
    flutter pub get
    ```
